Controles

W, A, S, D: Movimente o personagem
Enter: Interaja com os NPCs
ESC: Pausa o jogo

Objetivo

O objetivo do jogo é resolver o mistério do desaparecimento do diamante "O Olho de Windsor". Para isso, você precisará investigar as testemunhas e suspeitos e usar suas habilidades de programação SQL para encontrar as pistas necessárias.

Começando o jogo

O jogo começa com você no museu, onde o diamante foi roubado. Você precisará usar os comandos de movimento para explorar o museu e encontrar informações que o ajudem a resolver o mistério.

Interagindo com os NPCs

Você pode interagir com os NPCs pressionando Enter ao lado deles. Ao interagir com um NPC, você poderá fazer perguntas sobre o crime ou obter pistas.

Respondendo às perguntas

Ao longo do jogo, você precisará responder a perguntas sobre o crime. Para responder às perguntas, você precisará usar suas habilidades de programação SQL para pesquisar no banco de dados do museu.

Acertando as perguntas

Ao responder corretamente a 5 perguntas, você terá pistas suficientes para informar o suspeito.

Resolução do mistério

Quando você tiver coletado informações suficientes, poderá ir até o diretor do museu e informar quem você acha que está com o diamante. Se você estiver certo, você resolverá o mistério e vencerá o jogo. Se errar, você tem três chances de identificar o suspeito.

Dicas

Use a história para obter pistas.

Pense nas motivações dos suspeitos.

Use o banco de dados para pesquisar informações.

Boa sorte na investigação!
